## Intro

This is a project I started to play prediction games on Formula 1 with friends and family but also to have a dashboard and stats about F1 and ML based predictions. Dashboard is a fullstack app built with Nuxt 3 and stats/ml backend with FastAPI.

Checkout code repository [here](https://github.com/giraycoskun/f1-board)

Deployed at home server via docker-compose at ➜ [f1board.giraycoskun.dev](https://f1board.giraycoskun.dev)
